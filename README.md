# restaurante-monica
 Restaurante Mónica, localizado no coração do Porto no Shopping Brasília Piso -1, Loja 303  é um espaço acolhedor e que oferece uma experiência gastronómica única situado próximo à estação Campo 24 de Agosto, o restaurante proporciona fácil acesso para quem deseja saborear pratos tradicionais e  combinando qualidade, sabor e um ambiente confortável
